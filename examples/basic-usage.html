1. Single Image

This is the most basic usage. Apply pinch zoom to a single image using an ID selector. 

import { PinchZoom } from '../src/index.js';

const pinchZoom = new PinchZoom('#single-image');

2. Custom Options

Example with customized background color and maximum scale ratio.

const customZoom = new PinchZoom('#custom-image', {
    backgroundColor: 'rgba(0, 0, 0, 0.8)',
    maxScale: 4,
    transitionDuration: '0.5s'
});

3. Using Factory Function

Example using the createPinchZoom factory function.

import { createPinchZoom } from '../src/index.js';

const factoryZoom = createPinchZoom('#factory-image', {
    backgroundColor: 'rgba(255, 255, 255, 0.9)',
    maxScale: 3
});

4. Safe Initialization

Safe initialization method that doesn't throw errors.

import { safePinchZoom } from '../src/index.js';

const result = safePinchZoom('#safe-image', {
    backgroundColor: 'rgba(0, 100, 255, 0.6)'
});

if (result.success) {
    console.log('Initialization successful:', result.instance);
} else {
    console.error('Initialization failed:', result.error);
}
üí° Usage Tips:
  • On mobile devices, you can pinch images with two fingers to zoom in/out
  • On desktop, you can use mouse wheel (if supported by browser)
  • It's recommended to initialize PinchZoom after images are fully loaded
