🔍 PinchZoom 기본 사용법

1. 단일 이미지

가장 기본적인 사용법입니다. ID 선택자를 사용하여 하나의 이미지에 핀치 줌을 적용합니다. 

import { PinchZoom } from '../src/index.js';

const pinchZoom = new PinchZoom('#single-image');

2. 커스텀 옵션

배경색과 최대 확대 배율을 커스터마이징한 예제입니다.

const customZoom = new PinchZoom('#custom-image', {
    backgroundColor: 'rgba(0, 0, 0, 0.8)',
    maxScale: 4,
    transitionDuration: '0.5s'
});

3. 팩토리 함수 사용

createPinchZoom 팩토리 함수를 사용한 예제입니다.

import { createPinchZoom } from '../src/index.js';

const factoryZoom = createPinchZoom('#factory-image', {
    backgroundColor: 'rgba(255, 255, 255, 0.9)',
    maxScale: 3
});

4. 안전한 초기화

에러를 던지지 않는 안전한 초기화 방법입니다.

import { safePinchZoom } from '../src/index.js';

const result = safePinchZoom('#safe-image', {
    backgroundColor: 'rgba(0, 100, 255, 0.6)'
});

if (result.success) {
    console.log('초기화 성공:', result.instance);
} else {
    console.error('초기화 실패:', result.error);
}
💡 사용 팁:
  • 모바일 기기에서 두 손가락으로 이미지를 핀치하여 확대/축소할 수 있습니다
  • 데스크톱에서는 마우스 휠을 사용할 수 있습니다 (브라우저 지원 시)
  • 이미지가 완전히 로드된 후에 PinchZoom을 초기화하는 것이 좋습니다
